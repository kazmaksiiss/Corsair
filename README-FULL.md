## 📝 ЯК ВИКОРИСТОВУВАТИ:

### 1. Запуск сайту:
```
Відкрийте index.html у браузері
```

### 2. Додавання своїх фото:
```
1. Підготуйте фото (800x800px мінімум)
2. Помістіть в папку images/products/
3. Відредагуйте products-data.js
4. Змініть URL фото на свої
```

### 3. Додавання відео:
```
1. Відкрийте media-new.html
2. Знайдіть секцію "Ваше відео"
3. Замініть YouTube ID на свій
```

### 4. Редагування товарів:
```
1. Відкрийте products-data.js
2. Змініть назву, ціну, опис
3. Збережіть файл
```
